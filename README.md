## About

This repository documents my practical work and laboratory assignments for the **Computer Networks** course. It serves as a centralized record of the experiments, implementations, reports, and learning outcomes completed during the academic session.

The objective of this repository is to strengthen the understanding of networking concepts through practical experimentation and systematic documentation.

## Course Topics

- Introduction to Computer Networks
- Network Topologies
- OSI Reference Model
- TCP/IP Protocol Suite
- IPv4 & IPv6 Addressing
- Subnetting
- Routing and Switching
- TCP and UDP
- Socket Programming
- DNS, DHCP, HTTP, and FTP
- Network Security Fundamentals
- Packet Analysis using Wireshark
- Network Simulation using Cisco Packet Tracer
